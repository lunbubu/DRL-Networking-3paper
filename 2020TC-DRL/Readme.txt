continuousEnv.py: definition of the game environment, namely how mobile users and platform will interact with each other.

DNC.py: a simple version of DNC, referring to https://github.com/llSourcell/differentiable_neural_computer_LIVE.

PPO.py: an implementation of proximal policy optimization algorithm, referring to https://github.com/MorvanZhou/Reinforcement-learning-with-tensorflow.

DNC_PPO.py: an implementation of proximal policy optimization, where the value function contains DNC module for solving partially observable Markov decision process.

main.py: main function for this project.
